# Screenshot Checklist

These images are useful for the README and release page.

## Required

- `docs/screenshots/app-overview.png`: main app configured with real pool data.
- `docs/screenshots/widget-small.png`: small desktop widget showing the compact balance layout.
- `docs/screenshots/widget-medium.png`: medium desktop widget showing the `5h` and `Week` rings plus restore card.
- `docs/screenshots/widget-large.png`: large desktop widget showing quota rows and overall health status.
- `docs/screenshots/settings.png`: settings sidebar with Pool URL, Management key, refresh, plan weights, and widget options.

## Optional

- `docs/screenshots/health-timeline.png`: close-up of the app health timeline and account row request history.
- `docs/screenshots/no-data-debug.png`: widget/app message shown when App Group sync or fetching fails.
- `docs/screenshots/dmg-install.png`: DMG window with the app bundle before dragging to Applications.

## Privacy

Before publishing screenshots, hide or blur:

- Management key
- Pool URL if private
- Account names
- Auth indices
- Any internal server names or personal identifiers
